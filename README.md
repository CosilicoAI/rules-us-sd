# rules-us-sd

South Dakota statutes and regulations encoded in Cosilico DSL.

## Structure

```
rules-us-sd/
├── statutes/       # South Dakota Codified Laws (SDCL)
├── regulations/    # Administrative Rules of South Dakota (ARSD)
└── README.md
```

## Sources

- **Statutes**: [South Dakota Codified Laws](https://sdlegislature.gov/Statutes/Codified_Laws)
- **Regulations**: [Administrative Rules of South Dakota](https://sdlegislature.gov/Rules)

## Usage

This repository is part of the Cosilico ecosystem. Rules encoded here are used by `rac` (Rules as Code) for tax and benefit calculations.
